Penguins Data Analysis

This project performs Exploratory Data Analysis (EDA) on the Penguins dataset using Python. The analysis includes data cleaning, KPI calculations, correlation analysis, and multiple visualizations to understand penguin species characteristics and feature relationships.

Features
Data Cleaning
KPI Analysis
Species Distribution Visualization
Correlation Heatmap
Scatter Plot Analysis
Histogram & Pie Charts
Exploratory Data Analysis (EDA)
Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
